# TP - Crud File systeme

## Consigne

1. Creer un CRUD d'une entité POST 
2. stocker les données des postes dans un fichier `postData.json`
3. Utiliser le filesysteme module pour lire et modifier le fichier `postData.json`
4. Les routes a créer sont : 
    - GET collection 
    - GET detail
    - POST (creation)
    - PUT (modification)
    - DELETE (Deletion)
